## JSON Structure
Each color definition includes:
- Red Component (0.0-1.0)
- Green Component (0.0-1.0) 
- Blue Component (0.0-1.0)
- Color Space ("sRGB")

- Background Color
- Foreground Color
- Bold Color
- Selection Color
